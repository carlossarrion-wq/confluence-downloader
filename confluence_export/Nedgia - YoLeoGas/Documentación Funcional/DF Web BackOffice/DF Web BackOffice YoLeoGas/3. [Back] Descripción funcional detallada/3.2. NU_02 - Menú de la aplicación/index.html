El menú de la aplicación contiene las opciones correspondientes a las funcionalidades del rol del usuario que se ha autenticado.

Todas las que hay son:

  • Validación de lecturas

  • Validación de perfiles

  • Lecturas

  • Perfiles

  • Usuarios

  • Informes

  • Configuración

Además en la cabecera se mostrará:

  • a la izquierda, el logo de Nedgia y nombre de la aplicación que si se pulsa, lleva a la pantalla de inicio

  • a la derecha, el nombre y apellidos del usuario autenticado en la aplicación con su rol. También se muestra un desplegable desde donde se puede cerrar la sesión.
