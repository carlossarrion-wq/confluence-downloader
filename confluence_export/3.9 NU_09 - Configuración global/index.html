La Configuración Global establece los parámetros base del sistema que afectan a todas las distribuidoras, mientras que la Configuración de Distribuidoras permite personalizar estos parámetros para cada distribuidora específica, añadiendo funcionalidades avanzadas como notificaciones personalizadas, integración específica y configuraciones OCR más granulares.

Configuración Global (SuperAdmin):

  • Alcance: Afecta a todo el sistema a nivel global

  • Propósito: Parámetros generales que se aplican por defecto a todas las distribuidoras

  • Usuarios: Solo accesible por SuperAdmin

  • Tabla BD: configuration (registro único)

Configuración de Distribuidoras:

  • Alcance: Específica para cada distribuidora individual

  • Propósito: Personalización de parámetros por distribuidora, sobrescribiendo valores globales

  • Usuarios: Accesible por Manager y SuperAdmin

  • Tabla BD: distributor_configuration (múltiples registros, uno por distribuidora)

Funcionalidades Principales

Solo en Configuración Global:

  • Gestión de imágenes: Días de retención en Documentum

  • Horarios del sistema:

    • Hora de notificaciones generales

    • Hora de notificaciones de distribuidoras

    • Hora de informes de exceso

    • Hora de reindexación/mantenimiento BD

    • Hora de informes de backoffice

  • Configuración de email: Contraseña de cuenta de envíos

  • Configuración de sistema:

    • Habilitar valoraciones

    • Prioridad SCTD

    • Minutos entre reintentos de distribuidoras

    • Minutos entre limpiezas de Documentum

    • Meses para archivo histórico

Configuración de OCR

Algunos parámetros OCR existen en ambas configuraciones pero con diferente nivel de detalle:

Configuración Global (más básica):

  • Confianza mínima OCR móvil (1 valor general)

  • Confianza mínima OCR otras imágenes (valores básicos)

Configuración Distribuidoras (más granular):

  • Confianza por dígito restrictiva (8 valores)

  • Confianza por dígito tolerante (8 valores)

  • Configuración móvil restrictiva vs tolerante
